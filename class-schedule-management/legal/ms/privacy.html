Dasar Privasi

Terakhir dikemas kini: Oktober 2025

Dasar Privasi Aplikasi Pengurusan Jadual Kelas
Dasar ini menerangkan bagaimana kami mengumpul, menggunakan, dan melindungi maklumat peribadi anda apabila anda menggunakan aplikasi mudah alih kami.

1. Maklumat yang Kami Kumpulkan

1.1 Maklumat Peribadi

Kami mengumpul maklumat peribadi berikut apabila anda menggunakan aplikasi kami:

  • Nama dan alamat e-mel (melalui Google/Apple Sign-In jika anda berikan)
  • Foto profil/avatar (melalui Google/Apple Sign-In jika anda berikan)
  • Data jadual kelas
  • Maklumat pelajar/ahli yang anda masukkan
  • Data kursus dan sesi
  • Kemajuan pembelajaran dan keputusan
  • Tetapan dan pilihan

1.2 Maklumat yang Dikumpul Secara Automatik

  • Maklumat peranti (model, versi OS)
  • Analitik penggunaan aplikasi
  • Laporan kerosakan dan data diagnostik

1.3 Data Pengiklanan

Semasa menggunakan versi percuma dengan iklan dan jika anda bersetuju:

  • ID pengiklanan untuk iklan yang dipersonalisasi
  • Corak penggunaan aplikasi untuk sasaran iklan
  • Data klik dan interaksi dengan iklan

2. Bagaimana Kami Menggunakan Maklumat Anda

Kami menggunakan maklumat anda untuk:

  • Menyediakan perkhidmatan pengurusan jadual kelas
  • Menyegerakkan data merentasi peranti anda (Premium)
  • Meningkatkan fungsi aplikasi dan pengalaman pengguna
  • Menghantar pemberitahuan aplikasi penting
  • Sokongan pelanggan
  • Keselamatan dan pencegahan penipuan
  • Memaparkan iklan yang berkaitan
  • Memproses pembayaran langganan

3. Perkongsian dan Pendedahan Data

Kami TIDAK menjual maklumat peribadi anda.

Kami mungkin berkongsi data dengan:

3.1 Pembekal Perkhidmatan

  • Google/Apple untuk pengesahan
  • Perkhidmatan penyimpanan awan dan analitik
  • Perkhidmatan pengurusan langganan
  • Google AdMob untuk pengiklanan

3.2 Keperluan Undang-undang

Kami mungkin mendedahkan maklumat jika diperlukan oleh undang-undang atau untuk melindungi hak kami.

4. Penyimpanan dan Keselamatan Data

  • Data disimpan secara tempatan pada peranti anda dan secara pilihan dalam perkhidmatan awan
  • Semua penghantaran data menggunakan penyulitan (HTTPS/TLS)
  • Data tempatan dilindungi oleh ciri keselamatan peranti
  • Kami melaksanakan langkah keselamatan standard industri
  • Pengesahan biometrik disokong untuk capaian aplikasi

5. Hak Privasi Anda

Anda mempunyai hak untuk:

  • Mengakses data peribadi anda
  • Membetulkan maklumat yang tidak tepat
  • Memadam akaun dan data anda
  • Mengeksport data anda
  • Menarik diri daripada personalisasi pengiklanan

5.1 Pemadaman Data

Untuk memadam akaun anda dan semua data berkaitan:

  • Pergi ke Profil → Tetapan → Padam Akaun
  • Atau hubungi kami melalui e-mel di bawah

6. Privasi Kanak-kanak

Aplikasi kami ditujukan untuk pengguna berumur 13+ tahun. Kami tidak secara sedar mengumpul maklumat daripada kanak-kanak di bawah umur 13 tahun.

7. Pemindahan Data Antarabangsa

Data anda mungkin dipindahkan ke dan diproses di negara selain negara anda, termasuk Amerika Syarikat dan negara tempat pembekal perkhidmatan kami beroperasi.

8. Perkhidmatan Pihak Ketiga

Aplikasi kami bersepadu dengan:

  • Google Sign-In dan perkhidmatan Google
  • Apple Sign-In
  • Perkhidmatan awan (Google Cloud)
  • RevenueCat untuk pengurusan langganan
  • AdMob (Google) untuk pengiklanan

Sila semak dasar privasi mereka untuk amalan data mereka.

9. Pengekalan Data

Kami mengekalkan data anda:

  • Data akaun: Sehingga anda memadam akaun anda
  • Analitik penggunaan: Sehingga 24 bulan
  • Laporan kerosakan: Sehingga 12 bulan
  • Data pengiklanan: Mengikut dasar AdMob

10. Pematuhan Serantau

Untuk Pengguna Malaysia

Dasar ini mematuhi Akta Perlindungan Data Peribadi 2010 (PDPA).

Untuk Pengguna EU

Dasar ini mematuhi GDPR. Anda mempunyai hak tambahan di bawah Artikel 15-22 GDPR.

Untuk Pengguna California

Dasar ini mematuhi CCPA. Anda mempunyai hak tambahan di bawah California Consumer Privacy Act.

11. Kemas Kini Dasar Ini

Kami mungkin mengemaskini dasar privasi ini. Penggunaan berterusan aplikasi menunjukkan penerimaan perubahan.
