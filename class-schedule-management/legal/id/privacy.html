Kebijakan Privasi

Berlaku sejak: Februari 2026

Kebijakan Privasi Aplikasi Manajemen Jadwal Kelas
Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat menggunakan aplikasi seluler kami.

1. Informasi yang Kami Kumpulkan

1.1 Informasi Pribadi

Kami mengumpulkan informasi pribadi berikut saat Anda menggunakan aplikasi kami:

  • Nama dan alamat email (melalui Google/Apple Sign-In jika Anda berikan)
  • Foto profil/avatar (melalui Google/Apple Sign-In jika Anda berikan)
  • Data jadwal kelas
  • Informasi siswa/anggota yang Anda masukkan
  • Data kursus dan sesi
  • Kemajuan dan hasil belajar
  • Pengaturan dan preferensi

1.2 Informasi yang Dikumpulkan Otomatis

  • Informasi perangkat (model, versi OS)
  • Analitik penggunaan aplikasi
  • Laporan crash dan data diagnostik

1.3 Data Iklan

Saat menggunakan versi gratis dengan iklan dan jika Anda menyetujui:

  • ID Iklan untuk iklan yang dipersonalisasi
  • Pola penggunaan aplikasi untuk penargetan iklan
  • Data klik dan interaksi dengan iklan

2. Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan informasi Anda untuk:

  • Menyediakan layanan manajemen jadwal kelas
  • Menyinkronkan data di seluruh perangkat Anda (Premium)
  • Meningkatkan fungsionalitas aplikasi dan pengalaman pengguna
  • Mengirim notifikasi aplikasi penting
  • Dukungan pelanggan
  • Keamanan dan pencegahan penipuan
  • Menampilkan iklan yang relevan
  • Memproses pembayaran langganan

3. Berbagi dan Pengungkapan Data

Kami TIDAK menjual informasi pribadi Anda.

Kami dapat berbagi data dengan:

3.1 Penyedia Layanan

  • Google/Apple untuk autentikasi
  • Layanan penyimpanan cloud dan analitik
  • Layanan manajemen langganan
  • Google AdMob untuk iklan

3.2 Pemrosesan Pengembalian Dana

Ketika Anda meminta pengembalian dana untuk pembelian dalam aplikasi, kami dapat membagikan data penggunaan aplikasi Anda (seperti usia akun, riwayat pembelian, dan pola penggunaan) dengan Apple untuk membantu mengevaluasi permintaan pengembalian dana. Data ini membantu memastikan keputusan pengembalian dana yang adil. Dengan menggunakan aplikasi kami dan melakukan pembelian, Anda menyetujui pembagian data ini untuk tujuan pemrosesan pengembalian dana.

3.3 Persyaratan Hukum

Kami dapat mengungkapkan informasi jika diwajibkan oleh hukum atau untuk melindungi hak kami.

4. Penyimpanan dan Keamanan Data

  • Data disimpan secara lokal di perangkat Anda dan opsional di layanan cloud
  • Semua transmisi data menggunakan enkripsi (HTTPS/TLS)
  • Data lokal dilindungi oleh fitur keamanan perangkat
  • Kami menerapkan langkah-langkah keamanan standar industri
  • Autentikasi biometrik didukung untuk akses aplikasi

5. Hak Privasi Anda

Anda memiliki hak untuk:

  • Mengakses data pribadi Anda
  • Memperbaiki informasi yang tidak akurat
  • Menghapus akun dan data Anda
  • Mengekspor data Anda
  • Memilih keluar dari personalisasi iklan

5.1 Penghapusan Data

Untuk menghapus akun dan semua data terkait:

  • Pergi ke Profil → Pengaturan → Hapus Akun
  • Atau hubungi kami melalui email di bawah

6. Privasi Anak

Aplikasi kami ditujukan untuk pengguna berusia 13+ tahun. Kami tidak dengan sengaja mengumpulkan informasi dari anak-anak di bawah 13 tahun.

7. Transfer Data Internasional

Data Anda dapat ditransfer ke dan diproses di negara selain negara Anda sendiri, termasuk Amerika Serikat dan negara tempat penyedia layanan kami beroperasi.

8. Layanan Pihak Ketiga

Aplikasi kami terintegrasi dengan:

  • Google Sign-In dan layanan Google
  • Apple Sign-In
  • Layanan cloud (Google Cloud)
  • RevenueCat untuk manajemen langganan
  • AdMob (Google) untuk iklan

Silakan tinjau kebijakan privasi mereka untuk praktik data mereka.

9. Retensi Data

Kami menyimpan data Anda:

  • Data akun: Sampai Anda menghapus akun Anda
  • Analitik penggunaan: Hingga 24 bulan
  • Laporan crash: Hingga 12 bulan
  • Data iklan: Sesuai dengan kebijakan AdMob

10. Kepatuhan Regional

Untuk Pengguna Vietnam

Kebijakan ini mematuhi Undang-Undang Perlindungan Data Pribadi No. 91/2025/QH15, berlaku efektif 1 Januari 2026, dan Dekrit 13/2023/ND-CP tentang perlindungan data pribadi.

Hak Anda meliputi:

  • Persetujuan terpisah untuk setiap tujuan pemrosesan data (persetujuan terperinci)
  • Hak untuk mencabut persetujuan kapan saja
  • Hak untuk menolak pelacakan (Do Not Track)
  • Pemberitahuan dalam waktu 72 jam jika terjadi pelanggaran data

Untuk Pengguna UE

Kebijakan ini mematuhi GDPR. Anda memiliki hak tambahan di bawah Pasal 15-22 GDPR.

Untuk Pengguna California

Kebijakan ini mematuhi CCPA. Anda memiliki hak tambahan di bawah California Consumer Privacy Act.

Untuk Pengguna Korea Selatan

Kebijakan ini mematuhi PIPA (Personal Information Protection Act). Mulai Maret 2025, Anda memiliki hak untuk meminta portabilitas data dalam format yang aman dan dapat dibaca mesin.

Untuk Pengguna Malaysia

Kebijakan ini mematuhi PDPA (Personal Data Protection Act). Kami telah menunjuk Petugas Perlindungan Data dan akan memberi tahu Anda dalam waktu 72 jam jika terjadi pelanggaran data yang mempengaruhi Anda.

Untuk Pengguna Thailand

Kebijakan ini mematuhi PDPA (Personal Data Protection Act). Anda memiliki hak untuk mengakses, memperbaiki, menghapus, keberatan terhadap pemrosesan, membatasi pemrosesan, dan memindahkan data Anda.

Untuk Pengguna Indonesia

Kebijakan ini mematuhi UU PDP (Undang-Undang Perlindungan Data Pribadi). Anda memiliki hak untuk mengakses, memperbaiki, dan menghapus data pribadi Anda.

Untuk Pengguna Jepang

Kebijakan ini mematuhi APPI (Act on the Protection of Personal Information). Kami memerlukan persetujuan eksplisit Anda sebelum menempatkan cookie non-esensial.

Untuk Pengguna China

Kebijakan ini mematuhi PIPL (Personal Information Protection Law). Transfer data lintas batas mematuhi Langkah-langkah Sertifikasi Transfer Lintas Batas Informasi Pribadi, berlaku efektif 1 Januari 2026.

Untuk Pengguna Rusia

Kebijakan ini mematuhi Undang-Undang Federal 152-FZ. Data pribadi warga negara Rusia disimpan di server di Rusia. Persetujuan pemrosesan data diperoleh secara terpisah.
